# #024 「親プロセスと子プロセス」

```javascript
// fork-demo.mjs
import { fork } from 'node:child_process';
const worker = fork('./child.js');
console.log(`parent pid: ${process.pid}, child pid: ${worker.pid}`);
worker.on('exit', code => console.log(`child exited with ${code}`));
```
