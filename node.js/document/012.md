# #012 「イベントループとは何か」

```javascript
// event-loop-basic.mjs
console.log('start');
setTimeout(() => console.log('timer'), 0);
Promise.resolve().then(() => console.log('microtask'));
setImmediate(() => console.log('immediate'));
console.log('end');
```

```text
start
end
microtask
timer
immediate
```

// keep-alive.mjs
