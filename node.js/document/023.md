# #023 「プロセスIDとは」

```javascript
// pid-info.mjs
import process from 'node:process';
console.log({ pid: process.pid, ppid: process.ppid, execPath: process.execPath });
```

```bash
ps -o pid,ppid,cmd -p $(pgrep -n node)
```

```javascript
// worker-pid.mjs
import cluster from 'node:cluster';
if (cluster.isPrimary) {
  cluster.fork();
  cluster.on('online', worker => {
    console.log(`worker pid: ${worker.process.pid}`);
  });
} else {
  setInterval(() => {}, 1000);
}
```
