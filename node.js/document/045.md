# #045 「キャッシュミスとは」

```javascript
// cache-miss.mjs
const array = new Uint32Array(1024 * 1024);
console.time('stride-1');
for (let i = 0; i < array.length; i += 1) array[i]++;
console.timeEnd('stride-1');

console.time('stride-1024');
for (let i = 0; i < array.length; i += 1024) array[i]++;
console.timeEnd('stride-1024');
```

```bash
perf stat -e cache-misses,cache-references node cache-miss.mjs
```

```text
ストライド幅が大きいほどコンフリクトミス・キャパシティミスが発生しやすくなります。
```
