# #031 「メモリアドレスとポインタ」

```javascript
// buffer-address.mjs
const buf = Buffer.from('hello');
console.log({ byteOffset: buf.byteOffset, byteLength: buf.byteLength });
const view = new Uint8Array(buf.buffer, buf.byteOffset, buf.length);
console.log(view[0]);
```

```c
// addon.cc (抜粋)
Napi::Value RawPointer(const Napi::CallbackInfo& info) {
  Napi::Buffer<uint8_t> buffer = info[0].As<Napi::Buffer<uint8_t>>();
  printf("address: %p\n", buffer.Data());
  return info.Env().Undefined();
}
```

```javascript
// call-addon.mjs
import addon from './build/Release/addon.node';
addon.rawPointer(Buffer.allocUnsafe(16));
```
