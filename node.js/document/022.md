# #022 「ファイルディスクリプタとは」

```bash
ulimit -n    # 開けるファイルディスクリプタの上限を確認
```

```javascript
// fd-count.mjs
import fs from 'node:fs';
for (let i = 0; i < 100; i += 1) {
  fs.open(process.execPath, 'r', err => {
    if (err) console.error(err.code);
  });
}
```

```javascript
// graceful-close.mjs
import fs from 'node:fs';
const fd = fs.openSync('app.log', 'a');
process.on('exit', () => fs.closeSync(fd));
```
