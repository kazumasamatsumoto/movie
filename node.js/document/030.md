# #030 「スタック領域の管理」

```bash
node --stack_size=2048 deep-recursion.mjs
```

```javascript
// deep-recursion.mjs
function recurse(depth) {
  if (depth === 0) return 'done';
  return recurse(depth - 1);
}
console.log(recurse(1_000));
```

```javascript
// async-stack.mjs
async function chain(n) {
  if (n === 0) return 0;
  return 1 + chain(n - 1);
}
chain(1_000).catch(err => console.error(err));
```
