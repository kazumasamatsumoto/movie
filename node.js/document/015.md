# #015 「I/O Callbacksフェーズ」

```javascript
// io-callbacks.mjs
import fs from 'node:fs';
fs.readFile(new URL(import.meta.url), () => {
  console.log('fs.readFile handled in pending callbacks phase');
});

setTimeout(() => console.log('timer ready'), 0);
setImmediate(() => console.log('check ready'));
```
