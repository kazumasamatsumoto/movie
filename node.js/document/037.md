# #037 「プリエンプション」

```javascript
// preemption-demo.mjs
setInterval(() => console.log('event loop tick'), 100);

const heavy = () => {
  const start = Date.now();
  while (Date.now() - start < 200) {}
  setImmediate(heavy);
};
heavy();
```

sudo renice -n -10 $(pgrep -n node)
