# #068 「`const enum`を使ったらビルドが拒否」制作ドキュメント

## 台本
ずんだもん「TypeScript書いてるとさ、`const enum`を使ったらビルドが拒否って絶対あるよね？」
四国めたん「わかります。enumとリテラルどっち派か永遠に議論になりますよね」
ずんだもん「しかもそのまま`const enum`を使ったらビルドが拒否状態になって、チーム全員が固まるんだよ！」
四国めたん「そこでTypeScriptのリテラルvs enumの使い分けを使えば、`as const`とユニオンで軽量に表現できます」
ずんだもん「いや待って、それやると`const enum`を使ったらビルドが拒否第2ラウンドが始まるやつ！」
四国めたん「共感した方は“enum派閥”ってコメントで教えてください！」
ずんだもん「あなたのTypeScriptあるあるも募集中だよ！」

## 画面表示用コード

```typescript
enum RoleEnum {
  Admin = "admin",
  Editor = "editor",
}

const roleFromEnum: RoleEnum = RoleEnum.Admin;

const roles = ["admin", "editor"] as const;

type RoleLiteral = (typeof roles)[number];

const role: RoleLiteral = "admin";

function toLabel(role: RoleLiteral) {
  return role === "admin" ? "管理者" : "編集";
}

toLabel(role);
```
