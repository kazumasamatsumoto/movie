## 画面表示用コード

```typescript
type FeatureFlag = {
  mode: "strict" | "relaxed";
  retries: number;
};

const flag = {
  mode: "strict",
  retries: 3,
  debug: true,
} satisfies FeatureFlag; // debug が警告になる

// as だと気づかない
const loose = {
  mode: "strict",
  retries: 3,
  debug: true,
} as FeatureFlag;

function enable(flag: FeatureFlag) {
  return flag.mode.toUpperCase();
}

enable(flag);
```

## 制作メモ
- **想定シーン**: `satisfies`を導入して自信満々だったのに警告だらけになってしまう瞬間
- **狙いトーン**: ドヤ顔→膝から崩れ落ちる流れで笑わせる
- **技術キーワード**: `satisfies`, リテラル保持, 構造チェック, `as`との比較
- **使いたい一言**: 「satisfies勢」
- **CTA案**: コメントで“satisfiesが暴いたバグ”エピソードを募集
- **備考**: 画面には`debug`プロパティが弾かれる様子を映し、警告のありがたさと辛さを両立させる
