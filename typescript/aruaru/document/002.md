## 画面表示用コード

```typescript
// tsconfig.base.json
{
  "compilerOptions": {
    "strict": true,
    "module": "esnext"
  }
}

// tsconfig.app.json
{
  "extends": "./tsconfig.base.json",
  "include": ["src"],
  "exclude": ["dist"]
}

// package.json
{
  "scripts": {
    "typecheck": "tsc -p tsconfig.app.json --noEmit"
  }
}
```

## 制作メモ
- **想定シーン**: ローカルでは問題ないのにCIだけ落ちる夜中のビルドトラブル
- **狙いトーン**: パニック→冷静分析→再発フラグの流れで笑いを誘う
- **技術キーワード**: `tsconfig`, `extends`, `--noEmit`, CI
- **使いたい一言**: 「tsconfig地獄」
- **CTA案**: コメントで遭遇した設定事故やCI失敗談を募る
- **備考**: 設定ファイルの断片を画面に出して「結局ここか！」と共感させる
