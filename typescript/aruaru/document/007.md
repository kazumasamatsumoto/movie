## 画面表示用コード

```typescript
const roleLabel: Record<"admin" | "editor", string> = {
  admin: "管理者",
  editor: "編集者",
};

roleLabel.admin;

type FlagMap = Record<string, boolean>;

const flags: FlagMap = { feature: true };
flags.beta = false;

const typedFlags = {
  feature: true,
  beta: false,
} satisfies Record<string, boolean>;
```

## 制作メモ
- **想定シーン**: 汎用`Record<string, unknown>`に怯えながら適切なキー指定を探す夜のデバッグ
- **狙いトーン**: ため息まじりの愚痴をテンポ良く吐き出す
- **技術キーワード**: `Record`, リテラルキー, `satisfies`, マップ型
- **使いたい一言**: 「record地獄」
- **CTA案**: コメントで“辞書型の設計どうしてる？”と意見交換を促す
- **備考**: 辞書をリテラルで絞る例と汎用マップ両方を表示し、揺れる気持ちを表現する
