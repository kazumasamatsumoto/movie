## 画面表示用コード

```typescript
type Status = "draft" | "review" | "done";

function assertNever(x: never): never {
  throw new Error(`unexpected status: ${x}`);
}

function handleStatus(status: Status) {
  switch (status) {
    case "draft":
      return "WIP";
    case "review":
      return "Reviewing";
    case "done":
      return "Released";
    default:
      assertNever(status);
  }
}

handleStatus("draft");
handleStatus("done");
```

## 制作メモ
- **想定シーン**: `Pick`した型が`never`になって焦る瞬間に遭遇した瞬間を切り取るネタ
- **狙いトーン**: 理屈は冷静に、リアクションは大げさに
- **技術キーワード**: `never`, 判別ユニオン, 網羅性
- **使いたい一言**: 「never迷子」
- **CTA案**: コメントで「never迷子」と書いてもらい、同じ経験を募る
- **備考**: switch文での網羅性保証を短く扱うと伝わりやすい。
