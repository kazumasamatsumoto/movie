## 画面表示用コード

```typescript
const roles = ["admin", "editor"] as const;

function accept(role: (typeof roles)[number]) {
  return role.toUpperCase();
}

accept("admin");

const config = {
  mode: "dark",
  retries: 3,
} as const;

type Mode = typeof config.mode; // "dark"

function pickFlag<T extends readonly string[]>(flags: T) {
  return flags[0];
}

pickFlag(roles);
```

## 制作メモ
- **想定シーン**: 定数を定義したはずが`string`扱いに戻って警告が出る瞬間
- **狙いトーン**: ケアレスミスの悔しさをテンポ良く吐き出す
- **技術キーワード**: `as const`, リテラル型, 推論制御
- **使いたい一言**: 「asconst派」
- **CTA案**: コメントで“ここに`as const`付け忘れた”体験を募る
- **備考**: 配列・オブジェクト・関数引数の三連コンボで`as const`の効果を見せる
