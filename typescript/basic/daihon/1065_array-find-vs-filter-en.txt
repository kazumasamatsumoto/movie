ずんだもん「find stops as soon as it locates the first match.」
四国めたん「filter()[0] scans the entire array and then takes the first result, which is less efficient.」
ずんだもん「Both return T | undefined, but filter()[0] has more overhead.」
四国めたん「When you want a single element, choose find.」
