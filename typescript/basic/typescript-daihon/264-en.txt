四国めたん「Let's learn about falsy checks in TypeScript 」
ずんだもん「What does falsy mean?」
四国めたん「It refers to values that JavaScript evaluates as false.」
ずんだもん「Which values are falsy?」
四国めたん「false, 0, the empty string, null, undefined, and NaN.」
ずんだもん「Can we test it with an if statement?」
四国めたん「Yes—`if (!value)` checks whether the value is falsy.」
