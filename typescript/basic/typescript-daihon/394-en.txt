四国めたん「Its type is `void`; it doesn't alter the array—just applies side effects.」
ずんだもん「What about the callback's return?」
四国めたん「The callback is also `void`, so the return value is ignored; only the side effect matters.」
ずんだもん「I see! That's different from `map`!」
四国めたん「Exactly—`map` returns a new array, while `forEach` returns nothing, even in TypeScript v5.9.」
