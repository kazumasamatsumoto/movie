# #323 「デバッグ方法」

四国めたん「strictNullChecksのデバッグ方法を学びましょう!」
ずんだもん「まずはエラーメッセージをしっかり読むんだね!」
四国めたん「はい。Type 'string | null' is not assignableというエラーを確認します。」
ずんだもん「型ガードで修正するんだね?」
四国めたん「その通りです。nullチェックを追加して型を絞り込みます。」
ずんだもん「tsc --noEmitで型チェックだけできるんだね!」
四国めたん「はい。コンパイルせず型エラーだけを確認できます。」
ずんだもん「エラーを読んで、適切に修正していくのだ!」

---


```typescript
/** Example 1: エラーメッセージを読む */
let name: string = getValue();
// Error: Type 'string | null' is not assignable to type 'string'

/** Example 2: 型ガードで修正 */
const value = getValue();
if (value !== null) {
  const name: string = value; // OK
}

/** Example 3: 型チェックのみ実行 */
tsc --noEmit
// コンパイルせず型エラーだけ確認
```
