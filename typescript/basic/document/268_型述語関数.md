# #268 「型述語関数 - x is Type」

四国めたん「型述語関数について学びましょう!」
ずんだもん「value is stringという戻り値の型は普通のbooleanと違うの?」
四国めたん「はい。TypeScriptに型を絞り込むことを伝える特別な構文です。」
ずんだもん「関数がtrueを返すと、その値が指定した型だとTypeScriptが理解するんだね!」
四国めたん「その通りです。if文の中で型安全にメソッドを呼び出せるようになります。」
ずんだもん「isString()でチェックすれば、toUpperCase()が使えるようになるのか!」
四国めたん「unknown型の値を扱う時に特に便利で、型安全性が保たれます。」
ずんだもん「型述語を使えば、型ガードを自作できて柔軟な型チェックができるのだ!」

---


```typescript
/** Example 1: 文字列の型述語関数 */
function isString(value: unknown): value is string {
  return typeof value === 'string';
}

/** Example 2: 数値の型述語関数 */
function isNumber(value: unknown): value is number {
  return typeof value === 'number';
}

/** Example 3: 型述語の使用例 */
function process(value: unknown) {
  if (isString(value)) {
    console.log(value.toUpperCase());
  } else if (isNumber(value)) {
    console.log(value.toFixed(2));
  }
}
```
