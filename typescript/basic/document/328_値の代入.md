# #328 「値の代入」

四国めたん「nullable型への値の代入について学びましょう!」
ずんだもん「通常の値とnullの両方を代入できるんだね!」
四国めたん「はい。string | null 型には、文字列とnullのどちらも代入可能です。」
ずんだもん「オブジェクトもnullableにできるんだよね?」
四国めたん「その通りです。User | null には、Userオブジェクトかnullを代入できます。」
ずんだもん「配列もnullableにできるんだね!」
四国めたん「はい。string[] | null で、配列全体をnullableにできます。」
ずんだもん「nullable型なら、値の有無を柔軟に扱えるのだ!」

---

## 📺 画面表示用コード

```typescript
/** Example 1: 値の代入 */
let name: string | null = "Alice";
name = null; // OK
name = "Bob"; // OK

/** Example 2: オブジェクトの代入 */
let user: User | null = { name: "Alice", age: 25 };
user = null; // OK

/** Example 3: 配列の代入 */
let items: string[] | null = ["a", "b"];
items = null; // OK
items = ["c", "d"]; // OK
```
