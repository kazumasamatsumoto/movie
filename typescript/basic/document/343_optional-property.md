# #343 「オプショナルプロパティ - prop?」

四国めたん「オプショナルプロパティについて学びましょう!」
ずんだもん「プロパティ名の後に ? を付けるんだね!」
四国めたん「はい。そのプロパティを省略可能にできます。」
ずんだもん「オブジェクトリテラルで省略できるの?」
四国めたん「その通りです。必須プロパティだけ指定すれば、オプショナルは省略できます。」
ずんだもん「型エイリアスでも使えるんだね!」
四国めたん「はい。デフォルト値を使う想定のプロパティに適しています。」
ずんだもん「オプショナルプロパティで、柔軟なインターフェースを作るのだ!」

---

## 📺 画面表示用コード

```typescript
/** Example 1: オプショナルプロパティの基本 */
interface User {
  name: string;
  age?: number;       // 省略可能
  email?: string;     // 省略可能
}

/** Example 2: オブジェクトリテラルでの省略 */
const user1: User = { name: "Alice", age: 30 };
const user2: User = { name: "Bob" }; // ageとemailは省略

/** Example 3: 型エイリアスでの利用 */
type Config = {
  host: string;
  port?: number;      // デフォルト値を使う想定
  ssl?: boolean;
};
```
