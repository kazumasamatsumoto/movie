# #246 「大なり演算子 - >」

四国めたん「大なり演算子について学びましょう!」
ずんだもん「>を使うと、左側が右側より大きいかをチェックできるんだね!」
四国めたん「はい。数値の比較が基本ですが、文字列やDateオブジェクトにも使えます。」
ずんだもん「10 > 5はtrueだけど、5 > 5はfalseになるよね?」
四国めたん「その通りです。等しい場合は以上演算子>=を使います。」
ずんだもん「型安全な比較関数を作って、コードを読みやすくするのも大事だね!」
四国めたん「条件分岐で年齢チェックなどによく使われるので、しっかりマスターしましょう。」
ずんだもん「数値だけじゃなく文字列も辞書順で比較できるから便利なのだ!」

---


```typescript
/** Example 1: 基本的な使用例 */
console.log(10 > 5);   // true
console.log(5 > 10);   // false
console.log(5 > 5);    // false

/** Example 2: 型安全な比較関数 */
function isGreater(a: number, b: number): boolean {
  return a > b;
}

/** Example 3: 条件分岐での使用 */
const age: number = 20;
if (age > 18) {
  console.log('成人です');
}
```
