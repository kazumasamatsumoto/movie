# #321 「マイグレーション」

四国めたん「strictNullChecksへのマイグレーション手順を学びましょう!」
ずんだもん「まずはstrictNullChecksを有効化するんだね!」
四国めたん「はい。tsconfig.jsonでstrictNullChecksをtrueに設定します。」
ずんだもん「次に型定義でnull許容を明示的に指定するんだね?」
四国めたん「その通りです。string | nullのように明示します。」
ずんだもん「最後にnullチェックを実装するんだね!」
四国めたん「はい。型ガードやオプショナルチェーンを使って安全にします。」
ずんだもん「段階的に移行して、型安全なコードにするのだ!」

---


```typescript
/** Example 1: strictNullChecks有効化 */
{
  "compilerOptions": {
    "strictNullChecks": true
  }
}

/** Example 2: 型定義の追加 */
interface User {
  name: string;
  email: string | null; // null許容を明示
}

/** Example 3: nullチェックの実装 */
function getUser(id: number): User | null {
  const user = findUser(id);
  if (!user) return null;
  return user;
}
```
