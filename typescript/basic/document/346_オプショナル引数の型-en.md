Shikoku Metan: "Let's learn about optional parameter types!"
Zundamon: "When we add ?, it becomes string | undefined!"
Shikoku Metan: "Yes. Optional parameters automatically include undefined."
Zundamon: "What's the difference from default parameters?"
Shikoku Metan: "With default parameters, the type is pure string without undefined."
Zundamon: "Can we use optional for object types too?"
Shikoku Metan: "Yes. We can make the entire object optional, like options?: { verbose: boolean }."
Zundamon: "Let's create flexible functions with optional parameters!"

```typescript
/** Example 1: Optional parameters and undefinedable */
function log1(msg?: string) {
  // msg: string | undefined
  console.log(msg ?? "No message");
}

/** Example 2: Difference from default parameters */
function log2(msg: string = "default") {
  // msg: string (not undefined)
  console.log(msg);
}

/** Example 3: Combined with type inference */
function process(data: number, options?: { verbose: boolean }) {
  if (options !== undefined) {
    console.log(options.verbose);
  }
}
```
