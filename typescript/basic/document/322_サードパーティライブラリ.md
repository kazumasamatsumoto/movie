# #322 「サードパーティライブラリ」

四国めたん「サードパーティライブラリのstrictNullChecks対応を学びましょう!」
ずんだもん「型定義ファイルを作成するんだね!」
四国めたん「はい。types/legacy-lib.d.tsに型定義を作成します。」
ずんだもん「skipLibCheckを使うこともできるんだね?」
四国めたん「その通りです。一時的な対応としてskipLibCheckを使います。」
ずんだもん「型アサーションで対応する方法もあるんだね!」
四国めたん「はい。asを使ってstring | null型を明示的に指定します。」
ずんだもん「ライブラリの型定義を適切に管理するのだ!」

---


```typescript
/** Example 1: 型定義ファイル作成 */
// types/legacy-lib.d.ts
declare module 'legacy-lib' {
  export function getData(): string | null;
}

/** Example 2: skipLibCheck (一時的) */
{
  "compilerOptions": {
    "strictNullChecks": true,
    "skipLibCheck": true
  }
}

/** Example 3: 型アサーションで対応 */
import { getData } from 'legacy-lib';
const data = getData() as string | null;
```
