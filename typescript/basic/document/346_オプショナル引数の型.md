# #346 「オプショナル引数の型」

四国めたん「オプショナル引数の型について学びましょう!」
ずんだもん「?をつけると、string | undefinedになるんだね!」
四国めたん「はい。オプショナル引数は自動的にundefinedが追加されます。」
ずんだもん「デフォルト引数との違いは何なの?」
四国めたん「デフォルト引数の場合、型はundefinedを含まず純粋なstring型になります。」
ずんだもん「オブジェクト型のオプショナルも使えるの?」
四国めたん「はい。options?: { verbose: boolean } のように、オブジェクト全体をオプショナルにできます。」
ずんだもん「オプショナル引数で柔軟な関数を作るのだ!」

---

## 📺 画面表示用コード

```typescript
/** Example 1: オプショナル引数とundefinedable */
function log1(msg?: string) {
  // msg: string | undefined
  console.log(msg ?? "No message");
}

/** Example 2: デフォルト引数との違い */
function log2(msg: string = "default") {
  // msg: string (undefinedにならない)
  console.log(msg);
}

/** Example 3: 型推論との組み合わせ */
function process(data: number, options?: { verbose: boolean }) {
  if (options !== undefined) {
    console.log(options.verbose);
  }
}
```
