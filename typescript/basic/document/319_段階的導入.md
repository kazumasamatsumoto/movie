# #319 「段階的導入」

四国めたん「strictNullChecksの段階的導入方法について学びましょう!」
ずんだもん「ファイル単位で@ts-checkを使うと、部分的に有効化できるんだね!」
四国めたん「はい。大規模プロジェクトでも少しずつ対応できます。」
ずんだもん「tsconfig.jsonでファイルを指定する方法もあるんだよね?」
四国めたん「その通りです。filesプロパティで対象ファイルを限定できます。」
ずんだもん「tsc --noEmitでエラー数を確認するのが良いんだね!」
四国めたん「はい。全体のエラーをリスト化して、優先順位を付けて対応します。」
ずんだもん「段階的に導入することで、リスクを最小限にできるのだ!」

---


```typescript
/** Example 1: ファイル単位で有効化 */
// @ts-check
// strictNullChecks有効として扱う

/** Example 2: tsconfig.json - 段階的設定 */
{
  "compilerOptions": {
    "strictNullChecks": false
  },
  "files": ["src/new-module.ts"]
}

/** Example 3: エラー数を確認 */
tsc --noEmit
// エラーをリスト化して優先順位付け
```
