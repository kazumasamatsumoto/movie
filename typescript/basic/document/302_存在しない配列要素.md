# #302 「存在しない配列要素」

四国めたん「存在しない配列要素について学びましょう!」
ずんだもん「配列の範囲外にアクセスするとどうなる?」
四国めたん「はい。JavaScriptではundefinedが返されます。」
ずんだもん「arr[10]で配列の長さが3ならundefinedなんだね!」
四国めたん「その通りです。noUncheckedIndexedAccessで型安全にできます。」
ずんだもん「このオプションを有効にすると型がどう変わるの?」
四国めたん「はい。arr[0]の型がnumber | undefinedになります。」
ずんだもん「undefinedチェックしてから使えば、安全にアクセスできるのだ!」

---

## 📺 画面表示用コード

```typescript
/** Example 1: 配列の範囲外アクセス */
const arr = [1, 2, 3];
const value = arr[10];  // undefined

/** Example 2: noUncheckedIndexedAccess */
// tsconfig: "noUncheckedIndexedAccess": true
const arr: number[] = [1, 2, 3];
const value = arr[0];  // number | undefined

/** Example 3: 安全なアクセス */
const arr = [1, 2, 3];
if (arr[10] !== undefined) {
  console.log(arr[10].toFixed());
}
```
