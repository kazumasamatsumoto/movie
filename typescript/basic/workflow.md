# TypeScriptドキュメント作成ワークフロー

## 概要
extracted_codes.mdやextracted_codes-v2.mdから個別のドキュメントファイルを切り出し、台本ファイルを作成・更新する一連の作業手順。

## ファイル構成
- `typescript-document/extracted_codes.md` - 元データ（#246-280）
- `typescript-document/extracted_codes-v2.md` - 元データ（#281以降）
- `document/` - 切り出し後のドキュメント（日本語・英語）
- `daihon/` - 台本ファイル（日本語・英語）

## 参考ファイル
- `document/551_unknown.md` - フォーマットの参考サンプル

## 作業手順

### 1. ドキュメントファイルの切り出し
元ファイルから該当番号（例: #246）の内容を読み取り、以下の形式で新規ファイルを作成：

**ファイル名**: `document/[番号]_[トピック名].md`
例: `document/246_greater-than-operator.md`

**フォーマット**:
```markdown
# #[番号] 「タイトル」

四国めたん「[導入の説明]」
ずんだもん「[質問や反応]」
四国めたん「[詳しい説明]」
ずんだもん「[具体例への反応]」
四国めたん「[補足説明]」
ずんだもん「[実用的なポイント]」
四国めたん「[まとめや重要ポイント]」
ずんだもん「[最後の感想やポイント]」

---

## 📺 画面表示用コード

\`\`\`typescript
/** Example 1: [説明] */
[コード例1]

/** Example 2: [説明] */
[コード例2]

/** Example 3: [説明] */
[コード例3]
\`\`\`
```

### 2. 日本語台本の作成・更新
**ファイル名**: `daihon/[番号].txt`

**内容**: ドキュメントファイルの対話部分（四国めたんとずんだもんの会話）をコピー

### 3. 英語台本の作成
**ファイル名**: `daihon/[番号]-en.txt`

**フォーマット** (ゆっくりムービーメーカー用):
```
Shikoku Metan「[英訳]」
Zundamon「[英訳]」
Shikoku Metan「[英訳]」
...
```

**重要**: 英語版でも日本語の鉤括弧「」を使用すること（ゆっくりムービーメーカーで読み込むため）

### 4. 英語ドキュメントの作成
**ファイル名**: `document/[番号]_[トピック名]-en.md`

**内容**: 日本語ドキュメントを英訳
- タイトル: `# #[番号] "[英訳タイトル]"`
- 会話部分を英訳
- コメントも英訳（`/** Example 1: Basic usage */`など）
- コード内のコンソール出力も英訳

## 注意事項
1. extracted_codes-v2.mdは大きなファイルなので、offset/limitパラメータを使用して読み込む
2. 台本ファイルは既存の場合は更新、なければ新規作成
3. 551_unknown.mdのフォーマットを参考にする
4. 対話は8行程度（四国めたん4回、ずんだもん4回）
5. コード例は3つのExampleに分ける
6. **英語版台本も日本語の鉤括弧「」を使用する**（ゆっくりムービーメーカー用）
7. 台本の形式: `話者名「セリフ」`（コロンではなく鉤括弧）

## コマンド例（#246の場合）

```bash
# 1. ドキュメント切り出し（手動で内容を確認しながら作成）
# Read: typescript-document/extracted_codes.md
# Write: document/246_greater-than-operator.md

# 2. 台本更新
# Read: daihon/246.txt（既存の場合）
# Edit: daihon/246.txt

# 3. 英語台本作成
# Read: daihon/246-en.txt（既存の場合）
# Edit: daihon/246-en.txt

# 4. 英語ドキュメント作成
# Write: document/246_greater-than-operator-en.md
```

## 自動化のためのチェックリスト
- [ ] 番号を確認（例: 246）
- [ ] extracted_codes.mdまたはextracted_codes-v2.mdから該当部分を読み込み
- [ ] document/[番号]_[トピック].mdを作成
- [ ] daihon/[番号].txtを作成/更新（形式: `話者名「セリフ」`）
- [ ] daihon/[番号]-en.txtを作成/更新（形式: `Shikoku Metan「English text」`）
- [ ] document/[番号]_[トピック]-en.mdを作成
- [ ] 英語版台本の形式を確認（鉤括弧「」を使用しているか）
