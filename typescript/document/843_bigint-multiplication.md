## 📺 画面表示用コード

```typescript
/** Example 1: 基本乗算 */
const product = 123456789n * 987654321n;

/** Example 2: 累乗の代替 */
const squared = 1_000_000n * 1_000_000n;

/** Example 3: ループ計算 */
function multiply(values: bigint[]): bigint {
  return values.reduce((acc, cur) => acc * cur, 1n);
}
```
