# #1101 「sort()の型」

四国めたん「Array.prototype.sortの型も押さえておきましょう。」
ずんだん「sort(compareFn?: (a: T, b: T) => number): this って定義されてるね。」
四国めたん「はい、破壊的メソッドで、戻り値は同じ配列参照です。」
ずんだん「比較関数を渡さないと文字列比較になるので注意しよう。」
四国めたん「sortの型を理解して適切に使ってください。」
ずんだん「比較関数でカスタムソートしよう！」

---

## 📺 画面表示用コード

```typescript
const values = [3, 1, 2];

values.sort((a, b) => a - b);
```
