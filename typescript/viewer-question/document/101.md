## 台本
ずんだもん「`mapped types`で`as`句、使いこなせてる？」

四国めたん「`as`句を使えばキーを動的に変換できるので、型レベルでの命名変換が実現できます」
ずんだもん「現場だとAPI型からフロント用に変換する時に`as`句が大活躍して、型生成スクリプトが減るんだよ！」
四国めたん「みなさんは`as`句を活用していますか？」
ずんだもん「使いこなしテクニックもコメントで教えてね！」

## 画面表示用コード

```typescript
type Getters<T> = {
  [K in keyof T as `get${Capitalize<string & K>}`]: () => T[K]
};

type User = { name: string; age: number };
type UserGetters = Getters<User>;
// { getName: () => string; getAge: () => number }
```
