# #110 「`namespace`まだ使ってる？完全卒業？」制作ドキュメント

## 台本
ずんだもん「`namespace`まだ使ってる？完全卒業？」
四国めたん「v5.9でもnamespaceは使えますが、ES Modulesが主流になりました」
ずんだもん「ぼくは昔のコードでnamespaceを見かけるけど、新規プロジェクトでは使わないんだよね」
四国めたん「namespaceはグローバル汚染を防ぐ古い機能で、現在はES Modulesが推奨されます」
ずんだもん「現場だとレガシーコードの移行が大変で、namespaceを段階的に削除する作業が発生するんだよ！」
四国めたん「みなさんはまだ使っていますか？」
ずんだもん「移行状況をコメントで教えて！」

## 画面表示用コード

```typescript
// 古いnamespace
namespace Utils {
  export function format(s: string) { return s.trim(); }
}

// 現在推奨のES Modules
export function format(s: string) {
  return s.trim();
}
```

## 制作メモ
- **問いの切り口**: namespaceをまだ使っているか、完全に卒業したか
- **想定視聴者層**: TypeScriptのモジュールシステムに関心があるエンジニア
- **押さえる技術キーワード**: namespace, ES Modules, レガシーコード
- **キャラの立場**: 四国めたん=整理役 / ずんだもん=実装現場視点
- **コメントで聞きたいこと**: 使用状況や移行の進捗
- **補足メモ**: namespaceからES Modulesへの移行を紹介
