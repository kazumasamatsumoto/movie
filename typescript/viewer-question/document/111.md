# #111 「`module`と`namespace`の使い分け、整理できてる？」制作ドキュメント

## 台本
ずんだもん「`module`と`namespace`の使い分け、整理できてる？」
四国めたん「v5.9ではES Modulesが標準で、namespaceは歴史的な機能になりました」
ずんだもん「ぼくは新規コードでは全部moduleで書くけど、古いライブラリの型定義でnamespaceを見かけるんだよね」
四国めたん「moduleはファイル単位のスコープ、namespaceはグローバルスコープの整理に使われます」
ずんだもん「現場だとd.tsファイルでnamespaceが残ってて、移行計画を立てるのが大変なんだよ！」
四国めたん「みなさんは使い分けを整理できていますか？」
ずんだもん「考え方をコメントで教えて！」

## 画面表示用コード

```typescript
// module (推奨)
export const API_URL = "https://api.example.com";
export function fetchData() { /* ... */ }

// namespace (レガシー)
declare namespace Global {
  const VERSION: string;
}
```
