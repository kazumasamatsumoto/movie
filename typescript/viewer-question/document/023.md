## 画面表示用コード

```typescript
// package.json
{
  "scripts": {
    "dev:ts-node": "ts-node --esm src/index.ts",
    "dev:tsx": "tsx src/index.ts"
  }
}

import { readFileSync } from "node:fs";

console.log(readFileSync("package.json", "utf8").length);
```
