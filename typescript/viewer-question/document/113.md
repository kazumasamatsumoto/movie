# #113 「`reference`コメント、書いたことある？」制作ドキュメント

## 台本
ずんだもん「`reference`コメント、書いたことある？」
四国めたん「v5.9では`reference`コメントは特殊なケースに限定され、通常のimportが推奨されています」
ずんだもん「ぼくは型定義ファイルで他の定義を参照するときくらいしか書かないんだよね」
四国めたん「`/// <reference types="..." />`は主にグローバル型定義の参照に使いますが、モジュールではimportを使うべきです」
ずんだもん「型定義パッケージを作ってたとき、referenceコメントの使いどころを理解するのに時間かかったんだよ！」
四国めたん「みなさんは`reference`コメントを書いたことがありますか？」
ずんだもん「使ったケースがあったらコメントで共有してね！」

## 画面表示用コード

```typescript
// グローバル型定義ファイル（global.d.ts）
/// <reference types="node" />
/// <reference types="jest" />

declare global {
  const APP_VERSION: string;
}
```

```typescript
// 型定義パッケージでの使用例
/// <reference types="react" />

declare module 'my-library' {
  export function MyComponent(props: React.PropsWithChildren): JSX.Element;
}
```

```typescript
// 通常はimportを使う（推奨）
import type { MyType } from './types';

export function useMyType(value: MyType): void {
  // ...
}
```
