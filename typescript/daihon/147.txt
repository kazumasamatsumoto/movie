## 📺 画面表示用コード

```typescript
// べき乗の型推論
let base: number = 2;
let exponent: number = 3;
let result = base ** exponent; // number型と推論

// 型推論の例
let side = 5;          // number型
let area = side ** 2;  // number型と推論

// 実用的な例
let rate = 1.1;
let years = 3;
let finalAmount = 1000 * (rate ** years); // number型
```
