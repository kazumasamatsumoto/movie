## 📺 画面表示用コード

```typescript
// インクリメント
let count: number = 5;
count++; // 6 (後置インクリメント)
++count; // 7 (前置インクリメント)

// 実用的な例
let index: number = 0;
index++; // 1

let score: number = 100;
++score; // 101

// ループでの使用
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```
