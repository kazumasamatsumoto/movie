# #273 「[disabled]属性」台本

四国めたん「TypeScript v5.9の[disabled]属性について学びましょう！」
ずんだもん「[disabled]属性ってどう使うの？」
四国めたん「boolean値でボタンやフォーム要素の無効化を制御します」
ずんだもん「Angularでどう書くの？」
四国めたん「[disabled]="isDisabled"のようにプロパティバインディングします」
ずんだもん「動的に切り替えられる？」
四国めたん「はい、booleanプロパティを変更すれば即座に反映されます」
ずんだもん「フォーム制御が簡単だね！」

---

## 📺 画面表示用コード

```typescript
// [disabled]属性

@Component({
  selector: 'app-form',
  template: `
    <button [disabled]="isSubmitting">送信</button>
    <input [disabled]="!isEditable" />
  `
})
export class FormComponent {
  isSubmitting: boolean = false;
  isEditable: boolean = true;
  
  submit(): void {
    this.isSubmitting = true;
    // API呼び出し処理
  }
}
