## 📺 画面表示用コード

```typescript
// string型の基本宣言
let name: string = "Alice";

// 型安全性の例
name = "Bob";     // OK
// name = 123;    // エラー: Type 'number' is not assignable to type 'string'

// 文字列メソッドの型チェック
console.log(name.toUpperCase()); // "BOB"
console.log(name.length);        // 3
```
