# #151 「単項マイナス - -x」台本

四国めたん「TypeScript v5.9の単項マイナスについて学びましょう！」
ずんだもん「単項マイナスって何？」
四国めたん「数値の符号を反転する-演算子です」
ずんだもん「どうやって使うの？」
四国めたん「-xのように数値の前に-を付けます」
ずんだもん「Angularの開発でも使える？」
四国めたん「はい、符号の反転や絶対値計算に活用できます」
ずんだもん「単項マイナスで最高だよ！」

---

## 📺 画面表示用コード

```typescript
// 単項マイナス
let positive: number = 5;
let negative: number = -positive; // -5

let negative2: number = -10;
let positive2: number = -negative2; // 10

// 実用的な例
let price: number = 1000;
let discount: number = -price; // -1000

let temperature: number = 25;
let negativeTemp: number = -temperature; // -25
```
