## 📺 画面表示用コード

```typescript
// べき乗演算子
let base: number = 2;
let exponent: number = 3;
let result: number = base ** exponent; // 8

// 実用的な例
let side: number = 5;
let area: number = side ** 2; // 25 (正方形の面積)

let rate: number = 1.1;
let years: number = 3;
let finalAmount: number = 1000 * (rate ** years); // 複利計算
```
