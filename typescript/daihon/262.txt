ずんだもん「どうやってチェックするの？」

ずんだもん「!= nullって便利そう！」
四国めたん「はい、value != nullはnullとundefinedの両方を判定できます」

## 📺 画面表示用コード

```typescript
// null/undefinedチェック

// 厳密なチェック
function process(value: string | null | undefined) {
  if (value === null) {
    console.log('null');
  } else if (value === undefined) {
    console.log('undefined');
  } else {
    console.log(value.toUpperCase());
  }
}

// 両方をチェック
function handle(value: string | null | undefined) {
  if (value != null) {
    // nullでもundefinedでもない
    console.log(value.toUpperCase());
  }
}
