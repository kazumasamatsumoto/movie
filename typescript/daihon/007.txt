# #007 「string型とリテラル型の違い - "hello" vs string」台本

四国めたん「TypeScript v5.9のstring型とリテラル型について学びましょう！」
ずんだもん「リテラル型って何？」
四国めたん「特定の値のみを許可する型です。"hello"型は"hello"のみ許可します」
ずんだもん「string型とは違うの？」
四国めたん「string型は任意の文字列、リテラル型は特定の値のみです」
ずんだもん「どう使い分けるの？」
四国めたん「ステータス管理など、限られた値の場合はリテラル型が有効です」
ずんだもん「型安全性がより厳密になって最高だよ！」

---

## 📺 画面表示用コード

```typescript
// string型: 任意の文字列
let message: string = "hello";
message = "world"; // OK

// リテラル型: 特定の値のみ
let status: "loading" | "success" | "error" = "loading";
status = "success"; // OK
// status = "pending"; // エラー

// 実用的な例
function setTheme(theme: "light" | "dark") {
  console.log(`Theme: ${theme}`);
}
```
