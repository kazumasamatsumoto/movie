---

## 📺 画面表示用コード

```typescript
// 剰余演算子
let a: number = 17;
let b: number = 5;
let remainder: number = a % b; // 2

// 実用的な例
let number: number = 42;
let isEven: boolean = number % 2 === 0; // true

let seconds: number = 125;
let minutes: number = Math.floor(seconds / 60);
let remainingSeconds: number = seconds % 60; // 5
```
