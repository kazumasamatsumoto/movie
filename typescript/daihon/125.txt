## 📺 画面表示用コード

```typescript
// NaNとは
let nan: number = NaN;
let invalidResult: number = 0 / 0;

// 実用的な例
let parseResult: number = parseInt("abc"); // NaN
let mathResult: number = Math.sqrt(-1); // NaN

console.log(nan); // NaN
console.log(invalidResult); // NaN
console.log(parseResult); // NaN
```
