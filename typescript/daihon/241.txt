# #241 「等価演算子 - ==と===」台本

四国めたん「TypeScriptの等価演算子について学びましょう！」
ずんだもん「==と===の2種類があるんだよね？」
四国めたん「はい。==は緩い比較、===は厳密比較です」
ずんだもん「どう違うの？」
四国めたん「==は型変換を行ってから比較します」
ずんだもん「===は？」
四国めたん「===は型も含めて厳密に比較します」
ずんだもん「TypeScriptでは===を使うべきなんだね！」

---

## 📺 画面表示用コード

```typescript
// 緩い比較（==）
1 == "1"     // true（型変換が発生）
0 == false   // true
null == undefined  // true

// 厳密比較（===）
1 === "1"    // false（型が異なる）
0 === false  // false
null === undefined  // false

// TypeScriptでは===を推奨
const value: number = 42;
if (value === 42) {  // 型安全
  console.log("正しい比較");
}
