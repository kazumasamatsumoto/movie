ずんだもん「Are values forbidden?」
四国めたん「If the declared return type is `void`, returning a value is an error—except for `undefined`.」
ずんだもん「I see! Basically nothing comes back!」
四国めたん「Type inference handles it too; functions without a return are inferred as `void`.」
