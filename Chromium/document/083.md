# #083 「JavaScriptコードの実行フロー」

```text
1. Source text
2. Scanner -> Tokens
3. Parser -> AST
4. Ignition -> Bytecode
5. Bytecode execution with feedback
6. TurboFan -> Optimized code (if hot)
7. Deopt -> Back to Ignition (if guard fails)
```

```bash
node --print-bytecode script.js
node --trace-opt --trace-deopt script.js
```
