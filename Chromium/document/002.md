# #002 「ChromeとChromiumの違い完全解説」

```markdown
| 項目             | Chromium                    | Google Chrome                  |
|------------------|-----------------------------|--------------------------------|
| ライセンス       | BSD + OSSサブライセンス     | プロプライエタリ追加あり       |
| メディアコーデック| OSSコーデックのみ           | AAC / H.264 / Widevine対応     |
| アップデート     | 手動（ビルド or 配布依存）  | Google Update / omaha          |
| サービス連携     | なし                         | Google Sync, Translateなど     |
```

```bash
# Chromium起動
./out/Default/Chromium.app/Contents/MacOS/Chromium --no-first-run

# Chrome起動（macOS例）
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome
```

```text
Chromium: 検証・開発向け
Chrome: エンドユーザー向け安定版
```
