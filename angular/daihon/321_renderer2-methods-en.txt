# #321 「Renderer2 のメソッド」台本

ずんだもん「I want to review the representative Renderer2 methods!」
四国めたん「`setStyle`, `addClass`, `removeClass`, `setAttribute`, `removeAttribute`, and `listen` form the basic set.」
ずんだもん「Can it also add or remove elements?」
四国めたん「`createElement`, `appendChild`, and `removeChild` let you manipulate DOM nodes.」
ずんだもん「How do we remove events?」
四国めたん「Call the function returned from `listen` inside ngOnDestroy.」
ずんだもん「Any tips when combining it with Signals?」
四国めたん「Track changes and call only the necessary methods to avoid unnecessary repaints.」
