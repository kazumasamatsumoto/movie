# #323 「addClass() / removeClass()」台本

ずんだもん「How do we toggle classes with Renderer2?」
四国めたん「Use `addClass` and `removeClass` so you don't have to touch the DOM directly.」
ずんだもん「What if we want to switch based on conditions?」
四国めたん「Call the methods when Inputs change or apply the differences in ngOnChanges.」
ずんだもん「Is it okay to update multiple classes in a row?」
四国めたん「Yes. Call it one class at a time, or loop over an array if you want to batch it.」
ずんだもん「Does it work well with utility-first CSS like Tailwind?」
四国めたん「Yes. Wrapping conditional class assignment in a Directive keeps the template readable.」
