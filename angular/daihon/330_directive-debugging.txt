# #330 「Directive のデバッグ方法」台本

ずんだもん「Directiveをデバッグするときどうしてる？」
四国めたん「`ngDevMode`利用時にログを仕込むか、Angular DevToolsでホスト要素を追跡します。」
ずんだもん「HostBindingの値確認は？」
四国めたん「テンプレートに`{{ debugValue | json }}`を一時表示すると挙動を把握できます。」
ずんだもん「イベントハンドラの発火順が気になるな。」
四国めたん「Renderer2.listenの解除関数をSpyして、何回呼ばれているかテストで確認しましょう。」
ずんだもん「SSRでも再現したい時は？」
四国めたん「`npm run dev:ssr`でレンダリングし、ログやブラウザのHydration警告をチェックします。」
