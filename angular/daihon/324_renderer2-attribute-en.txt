# #324 「setAttribute() / removeAttribute()」台本

ずんだもん「How do we add or remove attributes with Renderer2?」
四国めたん「Call `setAttribute` to set a value and `removeAttribute` when it's no longer needed.」
ずんだもん「Can we handle accessibility attributes here?」
四国めたん「Absolutely. You can manage `aria-*` and `role` dynamically.」
ずんだもん「How do we express boolean attributes?」
四国めたん「Pass an empty string like `setAttribute(el, 'disabled', '')` to activate them.」
ずんだもん「Anything to watch out for performance-wise?」
四国めたん「Record the previous value so you avoid applying the same attribute repeatedly.」
