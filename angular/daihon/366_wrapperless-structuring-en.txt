# #366 「ラッパー要素なしの構造化」台本

ずんだもん「How do we organize structure without adding wrapper elements?」

ずんだもん「That prevents layout breaks, right?」
四国めたん「Yes. Since CSS stays untouched, it meshes well with existing styles.」
