# #094 「@Input() デフォルト値の設定」

ずんだもん「デフォルト値を設定する方法を教えて！」
四国めたん「@Input() propertyName: string = 'default'のように初期値を設定できます」
ずんだもん「複雑なオブジェクトでもできるの？」
四国めたん「はい！@Input() config: Config = {theme: 'light', size: 'medium'}のように設定可能です」
ずんだもん「デフォルト値があると親が値を渡さなくても安全だね！」
四国めたん「その通り！デフォルト値により、より柔軟で使いやすいコンポーネントAPIが実現できます」
