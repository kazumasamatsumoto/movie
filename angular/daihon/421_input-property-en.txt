# #421 「@Input() プロパティ」台本

ずんだもん「Show me how to write @Input().」
四国めたん「Declare it like `@Input() appHighlightColor = '#facc15';` so the template can supply a value.」
ずんだもん「Can we set an alias too?」
四国めたん「Yes-`@Input('appHighlight') color: string` lets you separate the attribute name from the property name.」
