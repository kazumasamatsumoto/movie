# #279 「UI Component Library の構築」台本

ずんだもん「UI Component Libraryを作るとき何から始める？」
四国めたん「デザインシステムのトークンを定義し、核となるボタンや入力コンポーネントを作ります。」
ずんだもん「検証はどう進めるの？」
四国めたん「Storybookでバリエーションを並べ、Visual Regressionテストを組み合わせます。」
ずんだもん「利用側への提供方法は？」
四国めたん「Standaloneエクスポートとスキーマドキュメントを整備し、npmパッケージ化します。」
ずんだもん「大規模開発でも安心して使えそうだね！」
四国めたん「ライブラリのガバナンスを決め、リリースフローを自動化すると定着します。」
