# #329 「Directive の命名規則」台本

ずんだもん「How do you handle Directive naming rules?」
四国めたん「Keep them consistent with an app prefix plus the feature name, like `appHighlightDirective`.」
ずんだもん「Is adding the Directive suffix mandatory?」
四国めたん「Add it to class names for clarity, and keep selectors short with verbs or adjectives.」
ずんだもん「What about file names?」
四国めたん「Standardize on dotted names such as `highlight.directive.ts`.」
ずんだもん「Do we align the test names too?」
四国めたん「Yes. Name them `highlight.directive.spec.ts` to make searching easier.」
