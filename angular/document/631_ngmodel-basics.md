# #631 「ngModel の基本」

## 概要
ngModelはテンプレート駆動フォームで入力とコンポーネントプロパティを連携させる基礎ディレクティブであり、値の同期と簡単なバリデーション制御を提供する。

## 学習目標
- ngModelの役割と動作を理解する
- テンプレートとコンポーネントの値同期の仕組みを学ぶ
- テンプレート駆動フォームの第一歩を押さえる

## 技術ポイント
- ngModelはFormsModuleをインポートすると利用可能
- テンプレートで[(ngModel)]やngModel単独で利用できる
- name属性と組み合わせてフォーム登録が可能

## 📺 画面表示用コード（動画用）
```html
<input [(ngModel)]="userName" />
```

## 💻 詳細実装例（学習用）
```typescript
protected userName = 'ずんだもん';

protected resetName(): void {
    this.userName = '';
}
```

## ベストプラクティス
- ngModelを使うコンポーネントではFormsModuleを確実に読み込む
- 初期値をコンポーネント側のプロパティで管理する
- 単純な入力からngModelの挙動を確認して理解を深める

## 注意点
- FormsModuleのインポートを忘れない
- ngModelだけではバリデーションが不十分な場合がある
- 複雑なロジックはリアクティブフォームへの移行を検討
