# #167 「:host-context セレクタ - 祖先条件」台本

ずんだもん「:host-contextって何に使うの？」
四国めたん「祖先要素に特定のクラスが付いているときだけスタイルを適用できます。例えば:host-context(.dark-theme) { ... } です」
ずんだもん「グローバルテーマを切り替えるのに便利そうだね？」
四国めたん「はい、親側でテーマクラスを切り替えると子コンポーネントのスタイルを連動できます」
ずんだもん「コンポーネントごとのカプセル化を保ちながら条件スタイルが書けるんだ！」
四国めたん「その通り、テーマやレイアウト条件に合わせた柔軟な適用が可能です」
