# #013 「ChromeOSとChromiumOSの関係」

```markdown
| 項目                | ChromiumOS                     | ChromeOS                            |
|---------------------|--------------------------------|-------------------------------------|
| ライセンス          | OSS (BSD系)                    | クローズド + 商用機能               |
| アップデート        | 開発者が制御                   | 自動更新(OMAHA)                     |
| Googleサービス      | なし                            | Google Play, Assistant, Drive統合   |
| 管理機能            | なし                            | Google Admin Console                |
```

```bash
# ChromiumOSソース取得
repo init -u https://chromium.googlesource.com/chromiumos/manifest.git
repo sync
```

```text
ChromeOS = ChromiumOS + Googleブランド／サービス／エンタープライズ機能
```
